# ClumsyBot
My Discord Bot

- Run first:
```bash
npm install
```
- To clear bot cmds run:
```bash
npm run clean
```
- or use docker.

create ```./config/.env```
```bash
# enter bot token here
# ex: TOKEN=AVCDEFGHIJ123456798
TOKEN=
# guild test id separated by comma with no space
# ex: TEST_IDS=123456789101112131,131211101987654321
TEST_IDS=
```
